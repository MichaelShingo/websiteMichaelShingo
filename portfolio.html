For the Lost Creative

For the Lost Creative by Creative Baggage is a curated database of free opportunities for musicians and artists. Inspired by a desire to find practical solutions to improve the lives of the creative community, the project saw its start after winning a grant from EnCORE Accelerando in Budapest, Hungary. Coded with Javascript and Velo by Wix. Opportunities populated using Python web scraping scripts.

CardMi

CardMi is a flashcard app that provides all the basic features necessary for effective study, including a minimalist environment for focus, the option to mark terms as studied, and sort and shuffle your deck to mix up your routine. Coded in Java with Android Studio.
